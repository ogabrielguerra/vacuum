What is it?

Vacuum is an experiment of turning Wordpress into a static site.

In fact, you don't need Wordpress to use Vacuum, just Json files.

All the main features you need

Metadata

SEO friendly with all the metadata you need.

Flexible

Easily add new properties editing static json files.

Titles, Categories and Tags

Just like you would have in Wordpress.

Media

Image handling and featured thumbnails.

Simple small footprint

PHP7 and only 2 main dependencies

Vacuum uses just two main dependencies: FastRoute and Twig.

Extensible

Of course you might extend it installing the library you want.

Fast but Powerful

Vacuum is intended to be fast, but it with all the power and flexibility of PHP.
